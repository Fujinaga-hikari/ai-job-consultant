## コンポーネント

### 共通クラス

#### `.input-field`

```css
w-full rounded-lg border border-gray-300 px-4 py-3 text-gray-800
transition-colors duration-200
focus:border-[#e63946] focus:ring-2 focus:ring-[#e63946]/20 focus:outline-none
```

- 角丸: `rounded-lg`（8px）
- パディング: `px-4 py-3`（横16px、縦12px）
- フォーカス時: ブランドカラーのボーダー＋薄いリング

#### `.btn-primary`

```css
rounded-full bg-[#ff7f7f] text-white font-bold
shadow-md transition-all duration-200
hover:bg-[#ff6666] hover:-translate-y-0.5 hover:shadow-lg
disabled:bg-gray-300 disabled:text-gray-500
```

- 形状: 丸角ボタン（`rounded-full`）
- ホバー: 上に0.5px浮き上がる＋影が強くなる
- 無効時: グレーアウト

#### `.article-box`（生成結果表示）

```css
bg-white rounded-xl border border-gray-100 p-6 md:p-8 leading-8 text-gray-700
```

- 行間: `leading-8`（2rem）で読みやすく

#### `.highlight-text`（強調テキスト）

```css
background: linear-gradient(transparent 60%, #ffdfdf 60%);
padding: 0 3px;
```

- 生成された Markdown の `**太字**` をマーカー風に装飾

## 画面構成

### メインページ（`/`）

```
┌─────────────────────────────────┐  max-w-3xl, mx-auto
│         [MixJob ロゴ]            │  width: 280px
│                                 │
│  ┌─────────────────────────┐    │
│  │   キャッチコピーバナー    │    │  rounded-2xl, border-gray-100
│  │  「無料でAIコンサル...」  │    │  ※ 結果表示中は非表示
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │   メインカード           │    │  bg-white, rounded-2xl, shadow-sm
│  │                         │    │  p-6 md:p-10
│  │  [入力フォーム]          │    │  ← 生成前
│  │    または               │    │
│  │  [生成結果 + 相談フォーム]│    │  ← 生成後
│  └─────────────────────────┘    │
│                                 │
│  フッター（コピーライト）         │
└─────────────────────────────────┘
```

**状態遷移**:

```
入力フォーム（JobForm）
  ↓ 生成ボタン押下・API 成功
生成結果（JobResult）＋ 相談フォーム（ConsultationForm）
  ↓ 「別の求人を作る」ボタン
入力フォームに戻る
```

### 入力フォーム（`JobForm`）

| ステップ | フィールド | 必須 |
|---------|-----------|------|
| 1 | 企業名 | ✅ |
| 2 | 募集職種名 | ✅ |
| 3 | 給与条件 | — |
| 4 | 勤務地 | — |
| 5 | 具体的な業務内容とミッション | ✅ |
| 6 | 理想のターゲット人物像 | — |

### 相談フォーム（`ConsultationForm`）

| フィールド | 必須 |
|-----------|------|
| メールアドレス | ✅ |
| お名前 | — |
| 会社名 | — |
| ご希望の連絡時間 | — |
| プライバシーポリシー同意チェック | ✅ |

## 管理画面（`/admin`）

### レイアウト

```
┌──────────────────────────────────────────┐
│ MixJob 管理 │ ダッシュボード │ 問い合わせ一覧 │ 生成ログ │ [ログアウト] │
├──────────────────────────────────────────┤
│                                          │
│  [ページコンテンツ]                        │
│                                          │
└──────────────────────────────────────────┘
```

- ナビ背景: `bg-white border-b border-gray-200`
- コンテンツ背景: `bg-gray-100`
- アクティブリンクホバー: `text-[#e84730]`

### ダッシュボード（`/admin`）

KPI カードのレイアウト:

```
── アクセス（PV / UU） ──────────────────────
 [累計 PV] [今月 PV] [今週 PV]   ← インディゴ border-l-4
 [累計 UU] [今月 UU] [今週 UU]   ← シアン border-l-4

── 問い合わせ ──────────────────────────────
 [累計]  [今月]  [今週]  [CV率]   ← ブランドレッド / グリーン

── AI求人生成 ──────────────────────────────
 [累計]  [今月]  [今週]           ← オレンジ

── グラフ ──────────────────────────────────
 [30日間トレンド（折れ線）]
 [週別推移（棒グラフ）]

── 直近5件の問い合わせ ──────────────────────
 [テーブル] [すべて見る →]
```

### KPI カードのスタイル

```css
bg-white rounded-xl shadow-sm p-5 border-l-4
/* border-left の色はカテゴリによって変化 */
```

## OG 画像仕様（`/opengraph-image`）

- サイズ: `1200 × 630 px`
- 生成: Next.js `ImageResponse`（動的生成、`opengraph-image.tsx`）
- 背景: `linear-gradient(135deg, #fff5f5, #ffffff, #fff8f0)`
- 上部アクセントライン: `linear-gradient(90deg, #e84730, #f2971b)`、高さ 8px
- 下部アクセントライン: 同グラデーション、高さ 4px
- ロゴ: SVG を Base64 で埋め込み、`480 × 120 px`
- キャッチコピー: `font-size: 36px, font-weight: 700, color: #1a1a1a`
- サブコピー: `font-size: 22px, color: #555`
- バッジ: 白背景 + `#e84730` ボーダー・テキスト、`border-radius: 999px`

| 要素 | アニメーション |
|------|--------------|

## レスポンシブ対応

| ブレークポイント | 対応内容 |
|----------------|---------|
| `sm`（640px〜） | KPI カードを 2列 → 3〜4列 |
| `md`（768px〜） | カードパディング `p-6` → `p-10`、キャッチコピーサイズ UP |
| `lg`（1024px〜） | キャッチコピー最大サイズ `text-3xl` |

最大幅: メインページ `max-w-3xl`（768px）、管理画面 `max-w-5xl`（1024px）/ `max-w-6xl`（1152px）
